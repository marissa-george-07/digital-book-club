# digital-book-club
A book discovery website with browsable collections, dynamic reviews, and a saved reading list — built with HTML, CSS, and JavaScript

## Features
- Browsable book collection grid with detail views for each book
- Dynamic review system — users can submit and view reviews instantly without reloading the page
- "Next Read" list powered by localStorage, so users can save books to read later
- Club sign-up form with input validation (name, email, age, genre)

## Tech Stack
- HTML
- CSS
- JavaScript (DOM manipulation, localStorage, form validation)
